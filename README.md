# My Ricing Configuration

This document lists the customization and configuration ("ricing") of my Linux system.

## Shell & Terminal
* **Shell**: Zsh
* **Prompt**: [Starship](https://starship.rs/)
* **Terminal Emulator**: Kitty, GNOME terminal
* **Shell Plugins**:
    * `zsh-autosuggestions`
    * `zsh-syntax-highlighting`
    * `zsh-autocomplete`
    * [Atuin](https://atuin.sh/) (Magical shell history)
* **System Info**: Fastfetch (Runs on shell startup)
* **Resource Monitor**: Btop

## Editor
* **Neovim**: Configured with [LazyVim](https://www.lazyvim.org/)
* **Font**: FiraCode Nerd Font
* **Plugin Manager**: Lazy.nvim
* **LSP & Autocompletion**: Configured via LazyVim
* **Additional Plugins**:
    * AI and completion
    > `Code Companion`
    > `Github Copilot`
    > `Github Copilot Chat (paid)`
    * Themes and visuals
    > `Themes`: `tokyonight`, `Black Metal Theme` (darkthrone)
    * Productivity
    > `Markdown Preview`
